# Expense Tracker

A lightweight personal expense tracker built with **Streamlit** and **SQLite**.

## Features

- Add expenses with date, amount, category, and description
- View all expenses in a sortable table
- Delete expenses by ID
- See monthly spending totals in a bar chart
- Persistent local storage via SQLite — no external database needed

## Tech Stack

- Python
- Streamlit (UI)
- SQLite (storage)
- Pandas (data handling)

## Getting Started

```bash
pip install streamlit pandas
streamlit run app.py
```
